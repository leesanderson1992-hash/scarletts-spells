# ADLE 7-UI Source Artifacts

This folder retains source artifacts used for 7-UI category design and teaching-content reconciliation.

The authoritative source-artifact register is:

- `source-artifact-register.csv`

Downloads or local attachment paths are not durable sources. Any source artifact used for planning or implementation must be copied here byte-for-byte and registered with checksum, intake date, source, status, transformation notes, validation artifact, and supersession status.
