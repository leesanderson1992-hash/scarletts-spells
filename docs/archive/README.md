# Docs Archive

## Purpose

This folder is the explicit archive destination for historical but useful documentation in the Scarlett's Spells repo.

Archived documents belong here when they are:
- superseded by a newer canonical contract, architecture doc, status doc, or implementation plan
- still useful as historical context, review evidence, migration history, or decision background
- no longer appropriate to present as active implementation guidance

## Rules

- archived docs are reference material, not active build instructions
- active indexes should not link archived docs as if they are current unless they are explicitly labeled historical
- archive placement should preserve useful history instead of hard-deleting it when the document still has lasting value
