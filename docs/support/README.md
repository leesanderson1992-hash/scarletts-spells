# Docs Support

## Purpose

This folder holds support artifacts used for QA, reset, regression, and verification workflows.

These files are not part of the active product canon.

## Current contents

- `sql/` for reset and verification SQL helpers
- `reward-source-of-truth-definitions.md` for reward data-source reference material that supports, but does not define, the active contracts

## Usage rule

Support files may be linked from implementation and QA docs, but they should not be treated as contracts, architecture truth, or active implementation plans.
